# Neon Air Canvas

An interactive web application that uses computer vision to turn your webcam into a canvas. Draw neon glowing trails in mid-air using your hand gestures! Built completely client-side using JavaScript and MediaPipe Hands.

## ✨ Features
* **Neon Glow Engine:** Beautiful fluid lines drawn with standalone canvas glow properties.
* **Camera Mirroring:** Intuitive mirroring mechanics so drawing feels exact and real-time.
* **Customization HUD:** In-app color pickers and brush radius thickness sliders.

## 🖐️ Gesture Commands
| Gesture | Action | Description |
| :--- | :--- | :--- |
| **☝️ One Finger (Index)** | **Draw** | Tracks your index finger tip to create neon lines. |
| **🖐️ Open Palm** | **Erase** | Activates an eraser orb over your hand coordinate. |
| **✊ Closed Fist** | **Pause** | Halts tracking so you can move your hand freely. |

## 🚀 Quick Start
1. Clone this repository or download the files.
2. Open `index.html` in a web browser using a local server framework (e.g., VS Code Live Server extension).
3. Grant webcam access permission when prompted and start drawing!
